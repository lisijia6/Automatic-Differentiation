AutoDiff.forward module#

class Forward(f, *variables)[source]#

Bases: object

Initialize forward mode AD. For detailed implementation see AutoDiff.forward.Forward.grad().

Parameters:

  • f (function object) – A callable function object, the \(f: \mathbb{R}^m \mapsto \mathbb{R}^n\) function

  • variables (integer or float or numpy array) – The input for f

Variables:

  • val (integer or float or numpy array) – The output of function f(*variables)

  • der (integer or float or numpy array) – The gradient f(*variables) wrt. variables i.e. \(\frac{\partial f(\text{variables})}{\partial \text{variables}}\)

  • output (Node class object or list of Node class objects) – AutoDiff.node.Node(); a single node for scalar function or a list of output nodes for vector function.

>>> from AutoDiff import Forward
>>> import numpy as np
>>> # Example 1: univariate scalar function
>>> x0 = 50
>>> def f0(x):
>>>     return 0
>>> g0 = Forward(f0, x0)
>>> g0.val
0
>>> g0.der
0.0
>>> ####
>>> # Example 2: multivariate scalar function
>>> x1 = [1, 2]
>>> def f1(x1, x2):
>>>     return x1 * x2
>>> g1 = Forward(f1, *x1)
>>> g1.val
2
>>> g1.der
array([2., 1.])
>>> ####
>>> # Example 3: univariate vector function
>>> x2 = 50
>>> def f2(x):
>>>     return [np.sin(x), np.cos(x)]
>>> g2 = Forward(f2, x2)
>>> g2.val
array([-0.26237485,  0.96496603])
>>> g2.der
array([0.96496603, 0.26237485])
>>> ####
>>> # Example 4: multivariate vector function
>>> x3 = [3, 4, 5]
>>> def f3(x1, x2, x3):
>>>     x4 = x1 + x2
>>>     return [np.exp(5 * x3), x2 ** 3 + x4, 2 * np.sqrt(x1) * x4]
>>> g3 = Forward(f3, *x3)
>>> g3.val
array([7.20048993e+10, 7.10000000e+01, 2.42487113e+01])
>>> g3.der
array([[0.00000000e+00, 0.00000000e+00, 3.60024497e+11],
    [1.00000000e+00, 4.90000000e+01, 0.00000000e+00],
    [7.50555350e+00, 3.46410162e+00, 0.00000000e+00]])

static grad(f, *variables)[source]#

Evaluate the full Jacobian in forward mode. This is the method that is used internally by AutoDiff.forward.Forward.__init__(). For each (scalar or multivariate) function f, use \(m\) passes with different seed vector \(\mathbf{e}\), where each natural basis \(\mathbf{e} \in \mathbb{R}^{m}\), and \(m\) is the number in variables.

Parameters:

  • f (function object) – A callable function object to perform differentiation on

  • variables (integer or float or numpy array or list of intergers or floats) – The input for variables of function f

Returns:

function evaluation at variable x, Jacobian, a single output node (scalar function) or a list of output nodes (multivariate) function) i.e. \(\frac{\partial f(\text{variables})}{\partial \text{variables}}\). Stack the gradient rows into the full Jacobian.

Return type:

Node class object or list of Node class objects

AutoDiff.node module#

class Node(value, derivative=1)[source]#

Bases: object

This is a class that implements the Dual numbers and dunder methods to overload built-in operators including negation, addition, subtraction, multiplication, true division, and power. The class also overloads numpy operators including square root, exponential, logarithm, sine, cosine, and tangent. Reflective operations are also included.

Parameters:

  • value (integer or float or numpy array) – The value of the Node, it can be an interger or float or a 1D numpy array for 1D problem, or a multi-dimensional numpy array for multi-dimensional problem

  • derivative (integer or float or numpy array) – The derivative of the Node, defaults to 1. It can be an interger or float or a 1D numpy array for 1D problem, or a multi-dimensional numpy array for multi-dimensional problem

Variables:

  • val (integer or float or numpy array) – The value of the Node, it can be an interger or float or a 1D numpy array for 1D problem, or a multi-dimensional numpy array for multi-dimensional problem

  • der (integer or float or numpy array) – The derivative of the Node, defaults to 1. It can be an interger or float or a 1D numpy array for 1D problem, or a multi-dimensional numpy array for multi-dimensional problem

  • parent (list) – A list of parent Nodes of the current Node

  • op (list) – A list of strings representing operations

  • v_index (integer) – An integer used to track variable index in visualization

>>> x1.val

>>> x1.der

>>> x1.parent
[]
>>> x1.op

AutoDiff.rnode module#

class RNode(value)[source]#

Bases: object

This is a class that implements the Reverse Nodes for reverse mode calculation of automatic differentiation and dunder methods to overload built-in operators including negation, addition, subtraction, multiplication, true division, and power. The class also overloads numpy operators including square root, exponential, logarithm, sine, cosine, and tangent. Reflective operations are also included.

Parameters:

value (integer or float or numpy array) – The value of the RNode, it can be an interger or float or a 1D numpy array for 1D problem, or a multi-dimensional numpy array for multi-dimensional problem

Variables:

  • val (integer or float or numpy array) – The value of the RNode, set using parameter value

  • der (integer or float or numpy array) – The parent of the RNode, defaults to None. It can be an interger or float or a 1D numpy array for 1D problem, or a multi-dimensional numpy array for multi-dimensional problem

  • parent (list) – A list of parent RNodes of the current RNode

>>> x1 = RNode(5)
>>> x1.val
5
>>> x1.der
>>> x1.parent
[]
